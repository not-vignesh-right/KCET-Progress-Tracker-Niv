# KCET-Progress-Tracker-Niv

Personal KCET 2026 study companion (Vite + React). Track daily sessions, catch-ups, streaks, and chapter confidence.

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Push this repo to GitHub.
2. In [Vercel](https://vercel.com), import the repository.
3. Use defaults: **Framework Preset: Vite**, **Build Command:** `npm run build`, **Output Directory:** `dist`.

`vercel.json` is included for SPA routing.
